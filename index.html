Agendamento Avisos Escritórios

Alô, #SangueLaranja. Tamo de volta!

Estamos tomando todas as providências para uma volta segura e saudável.
Para isso, elaboramos um protocolo de agendamento.

Agendamento

Avisos
Importantes!

  1. 01

    Os escritórios voltarão a ficar abertos no horário das 8h às 18h.

  2. 02

    O retorno ao escritório só é possível através do agendamento. Os #SangueLaranjas podem optar qual escritório ir, além de data e estação de trabalho demarcada com um número.

  3. 03

    Todos os consultores deverão usar máscaras dentro das acomodações dos escritórios.

  4. 04

    Nas mesas existem adesivos sinalizando onde os #SangueLaranjas devem sentar, para que haja o distanciamento necessário.

  5. 05

    Os escritórios só podem comportar até 40% dos consultores, conforme a legislação.

  6. 06

    Será medida a temperatura ao chegar no escritório.
